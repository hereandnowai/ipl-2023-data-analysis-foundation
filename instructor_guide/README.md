# Instructor Guide: Data Analysis to AI Foundations

Welcome to the teaching team! This guide helps you facilitate the 10-project curriculum.

## 1. Curriculum Strategy
This course is "Math-Light, Code-Heavy." We implement algorithms from scratch (regression, classification, clustering) so students understand the **Why** before using high-level libraries (Scikit-learn).

## 2. Key Challenges for Students
- **NumPy Shapes**: Students often struggle with `np.newaxis` or array dimensions.
- **Pandas Slicing**: `.loc` vs. `.iloc` can be confusing.
- **Sigmoid Intuition**: Why we don't use a straight line for classification.

## 3. Recommended Schedule
- **Days 1-2**: Projects 1, 2, 3 (The foundation).
- **Day 3**: Projects 4, 5, 6 (Clustering and Visuals).
- **Day 4**: Projects 7, 8, 9 (Special types of data).
- **Day 5**: Final Capstone.

## 4. Troubleshooting
Each project includes a `download_data.py`. If a link breaks, use any CSV from Kaggle with a similar structure.
