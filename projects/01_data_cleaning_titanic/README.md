# Project 01: Titanic Data Cleaning & Analysis

Welcome to your first project! In this project, you'll clean and analyze the Titanic dataset, the most famous dataset in data science.

## 🛠️ Project Tasks

1.  **Data Loading:** Use Pandas to read `titanic.csv`.
2.  **Exploration:** Find the number of rows, columns, and data types.
3.  **Missing Values:** Identify columns with missing data and decide how to fill/remove them.
4.  **Feature Selection:** Remove columns that don't help in survival analysis (e.g., Ticket ID).
5.  **Visualization:** Create a bar chart showing the survival rate by gender and class.
6.  **Insights:** Answer: Who were the most likely survivors?

## 📂 File Structure

- `concepts.ipynb`: Learn the basics of NumPy, Pandas, and Matplotlib.
- `notebook.ipynb`: Your step-by-step project implementation.
- `download_data.py`: Run this first to get the dataset.
- `dataset_info.md`: Descriptions of all columns.
- `assignment.md`: Extra tasks for practice.
