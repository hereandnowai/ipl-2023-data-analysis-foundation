# NLP Text Analysis Concepts

Natural Language Processing (NLP) helps computers understand human language.

### Key Concepts
1. **Tokenization**: Breaking sentences into words.
2. **Lowercasing**: Converting "AI" and "ai" to the same format.
3. **Sentiment**: Categorizing text as Positive or Negative.

### Manual Approach (No NLTK)
We will use Python's `.split()`, `.lower()`, and basic word lists to calculate sentiment scores manually!
