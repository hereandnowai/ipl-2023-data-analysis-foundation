# Section 0: Final Capstone - End-to-End Project

This is where all your skills come together!

### The Challenge
We want to combine Population data and GDP data to answer: **Does population growth directly drive GDP, or is it the other way around?**

### Steps
1. **Cleaning**: Missing country codes.
2. **Merging**: Joining two different tables using a shared key (`Country`, `Year`).
3. **Pivoting**: Reshaping rows into columns.
4. **Conclusion**: Summary of all learning.
