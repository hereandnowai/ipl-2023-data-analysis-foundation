# Recommendation System Concepts

A recommendation system helps users find products/media they might enjoy.

### Two Approaches
1. **Content-Based Filtering**: Recommend items similar to what you liked (e.g., you liked 'Star Wars', here is another Space Movie).
2. **Collaborative Filtering**: People who liked 'Star Wars' also liked 'Star Trek'.

### Project Implementation: Content-Based
We will recommend movies shared in the same genre categories. If a movie is "Action|Adventure", we find others with the same tags!
