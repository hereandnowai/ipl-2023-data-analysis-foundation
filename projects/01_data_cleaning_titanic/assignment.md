# Assignment: Titanic Data Analysis Extensions

Complete these tasks to deepen your understanding:

### Task 1: Fare Distribution (Beginner)
Use Matplotlib to create a histogram showing the distribution of passenger fares. 
*Hint: Use `bins` to make it readable.*

### Task 2: Family Size Feature (Intermediate)
Create a new column called `FamilySize` by adding `SibSp` and `Parch` together. 
Does having a larger family increase or decrease survival rates? Create a plot to show this.

### Task 3: Age Grouping (Intermediate)
Create a new column `AgeGroup` (e.g., Child < 18, Adult 18-60, Senior > 60).
Which age group had the highest survival percentage?

### Task 4: Embarked Survival (Intermediate)
Which port of embarkation (C, Q, or S) had the highest survival rate? 
Create a bar chart to visualize this.

### Task 5: Advanced Filtering (Advanced)
Filter the dataset for female passengers in the 1st class who paid more than the average fare. 
How many were there, and did they all survive?
